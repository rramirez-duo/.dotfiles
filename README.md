#Rueben's Dotfiles

I'm mostly on thinkpads with ubuntu + i3wm tiling window manager these days...

Installing on a new machine:
----------------------------

```bash
git clone git@github.com:ruebenramirez/.dotfiles.git ~/.dotfiles
cd ~/.dotfiles
make ubuntu
```
